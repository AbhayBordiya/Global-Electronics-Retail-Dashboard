# 📊 Global Electronics Retail Dashboard

## About
Power BI dashboard analyzing 62,000+ retail 
transactions across 9 countries.

## Tools Used
Power BI | DAX | Power Query | Data Modelling

## Key Features
- Star Schema with 5 tables
- 14 DAX measures (YTD, MTD, YoY, MoM)
- 8 interactive visualizations
- Dynamic slicers and conditional formatting
